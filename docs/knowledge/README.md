# Background knowledge: "507 / Besyüzyedi" daily charts & reports newsletter

**Source:** Gmail subscription from `507yedi@gmail.com` (sender name *Besyüzyedi* / "E."), an X (Twitter)
`@e507` "Abonelikler"/superfollows author. Near-daily Turkish-language digest of economic & finance
**infographics** ("Bugün gözüme çarpan grafikler…") plus links to that day's curated research **reports**.

**Purpose of this folder:** a durable, version-controlled **knowledge base** distilled from those emails,
to mine for context and cross-references when generating insights. **This is explicitly NOT a data source
for any dashboard** — it never feeds `data/`, D1, or R2. It's background reading only.

**Coverage in this Gmail account:** 2026-01-30 → 2026-06-06 (the author says the regular subscription
began 2026-01-15). Emails land under a dedicated Gmail label, not just the inbox.

---

## Two eras

1. **Link-era (2026-01-30 → ~2026-05-04):** each email body is essentially *"today's reports: <X link>"* +
   boilerplate. The substance sits behind **X subscriber-only** posts that can't be fetched here. All issues
   are catalogued in [`link-era-archive.md`](link-era-archive.md) (date + subject + report-thread link).

2. **Commentary-era (~2026-05-05 → present):** the author moved the **infographics + commentary inline**.
   These carry real, quotable text (chart captions, macro takes) and end with that day's report links.
   Full-text digests live in [`issues/`](issues/), one file per issue.

> ⚠️ The infographics themselves are **images** — not OCR'd here. Each digest preserves the author's verbatim
> commentary and the `[image: …]` caption markers that show where each chart sat. To see a chart, open the
> email (thread IDs are in each file's front-matter).

## How the report series is numbered

The author numbers each shared report (e.g. `#328 QNB`, `#340 Goldman Sachs: Turkish Banks`, `#350 Capgemini`).
Numbers run roughly #320s (mid-May) → #350 (early June). The reports themselves are PDFs/threads behind the
`@e507` subscriber links recorded in each digest and in the archive — useful as a **reading list of which
houses published what, when**, even though the PDFs aren't retrievable here.

Recurring Turkey-relevant sources worth watching for your BDDK work: **Haluk Bürümcekci** (banks' liquid FX,
TL valuation), **TCMB** (FX demand, reserves, Financial Stability Report, rent index), **Goldman Sachs Turkish
Banks**, and the local brokers (İş Yatırım, Şeker, QNB, BNP, TÜYİD BİST trends).
