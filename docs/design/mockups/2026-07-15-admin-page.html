What dies

  • The boxed Card / Stat tiles and pill badges — the last legacy evidence-layer surface on the site.
  • "Freshness per source, against expected refresh cadence" — the age-vs-cadence framing that cried "stale" in the weeks between monthly releases.
  • The three-deep scroll of equal-weight panels with no summary.

What survives, unmoved

  • All six sources, the coverage matrix + per-cell re-extract, every pipeline trigger, traffic, the PDF deck — same data, same actions.
  • The password gate, sign-out, and purge-cache — restyled, not removed.
  • Fresh / Late / Stale still read green / amber / red — semantic state, not the blue accent.

What's new

  • The six sources become the vitals band — the one bold element; everything else goes quiet.
  • A computed status Read leads: what's current, what's not, in one sentence.
  • Freshness reads BDDK's real answer — "June not yet published" — not a due-date guess.
  • Controls are mono-caps with an ink underline; blue is reserved for links that navigate.
